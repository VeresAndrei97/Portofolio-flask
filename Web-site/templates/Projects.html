  Robotic Controller using a Depth Camera

In this project I tried to implement a robotic system that can replicate the movement of the right hand of a user in real-time. For this project I've needed 3 components:
  • Depth Camera
  • Robotic Arm
  • Personal laptop
    • In order to make the communication between the 3 components I've used ROS as main framework. Other softwares used:
  • C/C++
  • LINUX
  • Python
  • XML
  • Gazebo
    • This project represents also my Bachelor's Degree thesis.



      Web-Scrapping Stock Market

    The following project was made with the idea of making stock-market buys/sells automated.
    This application tracks in real-time the stock price for Tesla shares. It can notify you when the stock price has decreased with a certain percentage or when the price has increased and you should sell.
    For this project Python with its libraries such as Numpy, BeautifullSoup, Pandas, matplot come in great help. For future development the application will also make the transaction for you.
    This project can be seen on my Git account.
